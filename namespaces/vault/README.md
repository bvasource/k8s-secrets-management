# Vault

## How to deploy

- Run `cd ../vault`
- Run `make namespace-create`
- Run `make vault-deploy`
- Run `make vault-port-forward`
- Run `make vault-setup`

---

- Now Vault set, go into [dev-setup](../dev-setup/README.md)
